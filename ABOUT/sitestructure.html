COMPOSITION


I am currently devising official site structure templates for the different types of documents on my site. For each type of document listed below, you can click the header to show the standard code for each one (WIP). Currently, there are an overarching 5 different types:

HOME PAGE

The index, home of the Creation. Subtle differences in the index from other pages, like larger logo, different H1 header, and a slightly different table than link pages.

LINK PAGE

a transition page containing, you guessed it, links, to related pages. A table of contents for different sections of this repository. (ex, The Notebook)

ESSAY PAGE

A content rich page, often just text with an optional one image. One header + sub-header, and text. (ex, Essay on Minimalism)

CONTENT PAGE

An alternative content page, with more mixed media than an essay. Often contains a mix of images, videos, imbeded documents, lists, links, etc. (ex, this page!)
